# Решение в рамках хакатона - KION

MVP решения, которое:

1. Делит одночасовое видео на логически завершенные сцены (учитывая сюжетный смысл, а не монтаж).
2. Производит идентификацию и трекинг лиц на всем протяжении видео, несмотря на смену камер, освещения, ракурсов и так далее.

Для анализа был взят фильм и трейлер 'Home alone' 

Финальное решение включает в себя:
- выделение аудио и видео
- анализ аудио и видео для определения сцен (пиковая аудиоактивность, тишина, интонации)
- анализ темпа монтажа, длительности кадров, переходов
- интеллектуальное объединение сцен на основе аудио и видео весов в смысловые блоки
- сценарная сегментация
- детекция и трекинг персонажей

## Структура проекта
1. Сборка проекта:
    1. `build.sh` - файл для сборки образа
    2. `Dockerfile` - для сборки основного образа
    3. `docker-compose.yml` - для сборки контейнера
    4. `requirements.txt` - все требцемые модцли
2. `pipline.ipynb` - полный пайплайн решения
3. `presentation.pdf` - презентация работы
4. `Христофорова_П_А_292405-2_отчет о технологической практике.docx` - отчет по практике

## Сборка проекта
Для сборки достаточно запустить `./build.sh`, предварительно выдав ему права на исполнение. Все необходимые модули, пакеты и зависимости установятся автоматически

В результате будет собран контейнер:
- На Python 3.10
- С cuda 12.8 и поддержкой новой архитектуры sm120
- С доступом к мощностям GPU и юсером с правами на гит
- С созданным ядром Python 3.10 (kion) и поддержкой работы в jupyter 

Далее, можно тестировать в любой современной IDE, подключившись к запущенному контейнеру
